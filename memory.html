memori komputer adalah perangkat keras (hardware) yang digunakan untuk menyimpan data berupa informasi atau instruksi yang bersifat sementara maupun permanen. Memori merupakan kumpulan dari storage units yang menyimpan data binary dalam bentuk bit. Setiap blok memori terdiri dari sejumlah kecil komponen, bagian-bagian kecil ini disebut sel.Berada di dalam central processor unit (CPU), memori menjadi komponen penting untuk mengoperasikan perangkat komputer. Setiap data dan program yang diproses oleh processor semuanya akan disimpan pada memori. Data yang bersifat sementara akan hilang secara otomatis jika perangkat dimatikan. Sedangkan data yang bersifat permanen akan tetap ada sekalipun aliran listrik dimatikan.


Jenis Memory

Jenis Memory

Ram

RAM merupakan salah satu memori primer atau memori utama yang dapat diakses langsung oleh CPU secara cepat. RAM sendiri tergolong sebagai memori volatile sehingga data yang ada di dalamnya bisa hilang jika aliran listrik dimatikan. Oleh sebab itu, perangkat ini sering disebut sebagai media penyimpanan data sementara.

Rom

Sama halnya RAM, perangkat ini dapat diakses secara langsung oleh CPU dan termasuk memori utama. Namun, ROM memiliki sifat non-volatile sehingga data akan tetap tersimpan meski tidak ada daya sekalipun. Sebagai memori read-only, data ROM hanya dapat dibaca alias tidak bisa dimodifikasi.

Static Memory

Static memory, atau SRAM (Static Random Access Memory), adalah jenis memori yang mempertahankan data tanpa perlu penyegaran terus-menerus. Ini biasanya digunakan dalam cache memori dan register. Static memory cenderung lebih cepat daripada dynamic memory, tetapi juga lebih mahal.

Dynamic Memory

Dynamic memory, atau DRAM (Dynamic Random Access Memory), adalah jenis RAM yang memerlukan penyegaran periodik untuk mempertahankan data. Ini adalah jenis RAM yang paling umum digunakan dalam komputer karena biayanya yang lebih rendah, meskipun kurang cepat daripada static memory.

Volatile

Memori volatile dalam konteks komputer mengacu pada jenis memori yang dapat kehilangan isinya ketika daya listrik terputus. Dua bentuk memori utama yang memperlihatkan sifat ini adalah RAM (Random Access Memory) dan cache memory. Ketika komputer dinyalakan, data disimpan dalam RAM untuk memberikan akses cepat kepada CPU selama eksekusi program. Namun, saat daya dimatikan, data dalam RAM hilang, sehingga RAM dianggap sebagai memori volatile.

Non Volatile

Memori non-volatil merujuk pada jenis memori yang dapat menyimpan informasi atau data tanpa memerlukan daya listrik yang terus menerus. Ini berbeda dengan memori volatil, seperti RAM, yang kehilangan data ketika daya listrik diputus.Contoh umum dari memori non-volatil adalah hard drive (HDD), solid-state drive (SSD), dan flash drive. Ketiganya adalah perangkat penyimpanan yang dapat menyimpan data dalam jangka panjang tanpa memerlukan daya listrik saat tidak digunakan. Ini membuat mereka cocok untuk menyimpan data yang perlu dipertahankan bahkan ketika daya listrik dimatikan.
